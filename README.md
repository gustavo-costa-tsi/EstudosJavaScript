# ⚡ Estudos de JavaScript

Este repositório contém exercícios práticos focados em lógica de programação e fundamentos da linguagem.

## 🛠️ Tecnologias e Conceitos

* [x] **Variáveis:** Declaração, escopo e constantes.
* [x] **Tipos e entrada de dados:** Manipulação de tipos primitivos e entrada de dados.
* [ ] **Funções:** Declaração, parâmetros e arrow functions.
* [ ] **Operadores:** Relacionais (comparação) e lógicos.
* [ ] **Loops:** Estruturas de repetição.
* [ ] **Condicionais:** Estruturas de controle de fluxo.
* [ ] **Arrays:** Métodos de manipulação de listas.

## 🚀 Como executar

Os scripts devem ser executados via terminal utilizando o **Node.js**:
